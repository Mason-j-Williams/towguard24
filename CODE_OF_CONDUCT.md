Code of Conduct
TowGuard24 is committed to maintaining a safe, respectful, and professional environment for all contributors, users, and collaborators. This Code of Conduct outlines expectations for behavior and the process for reporting unacceptable conduct.

1. Our Standards
We expect all participants to:

Treat others with respect and professionalism

Use inclusive and welcoming language
Provide constructive feedback

Collaborate in good faith

Respect differing viewpoints

Follow project guidelines and maintain integrity
2. Unacceptable Behavior
The following behaviors are strictly prohibited:

Harassment, bullying, or intimidation

Hate speech or discriminatory remarks

Threats or violent language

Posting sexually explicit or inappropriate content

Personal attacks or insults

Disruptive or hostile behavior

Sharing private information without consent

Any illegal or unethical activity.
3. Responsibilities of Maintainers
Project maintainers are responsible for:

Clarifying standards of acceptable behavior

Taking appropriate action in response to violations

Removing or editing harmful content

Temporarily or permanently banning offenders when necessary

Maintainers must enforce this Code fairly and consistently.
4. Reporting Violations
If you witness or experience unacceptable behavior, report it privately:

Email:  
conduct@towguard24.com

Include:

Description of the incident

Names or usernames involved

Date, time, and location (if applicable)

Any supporting evidence

All reports will be handled confidentially.
5. Enforcement
TowGuard24 may take any action deemed appropriate, including:

Warning the offender

Temporary suspension

Permanent ban from the project

Removal of contributions

Escalation to appropriate authorities if required

6. Scope
This Code applies to:
All TowGuard24 repositories

Issue trackers

Pull requests

Discussions

Community spaces

Any interactions representing the project
